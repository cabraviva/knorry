- 🏭 Supports most features

- 🪶 As lightweight as possible
- ⚡ Pure ESM
- 📦 Optimized for bundlers
- 👨‍💻 Nice developer experience
- 🚀 Extremely good backwards [compatibility](https://caniuse.com/xhr2)
